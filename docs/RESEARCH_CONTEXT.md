# Research Context — Caesar AI Scan

A CLI and GitHub Action that finds AI usage in code and generates governance evidence gaps.

## Product type

- CLI
- GitHub Action
- scan rules
- Markdown/HTML report
- evidence export

## Main users

- developers
- CTOs
- security teams
- compliance teams
- Caesar consulting clients

- find OpenAI/Anthropic/Gemini SDKs
- find LangChain/LlamaIndex/agent frameworks

- find vector DBs
- find n8n AI workflow exports
- generate evidence gaps
