# Local SQL Compile Rehearsal Boundary

This documentation outlines the local, offline-only validation process for SQL schemas, ensuring consistency and safety for the Supabase migration rehearsal.

- **Status:** Experimental / Prototype
- **Nature:** Local, Static, Offline-only
- **Scope:** Processes SQL files locally.
- **Constraints:** This does not interact with any production backend systems, remote Supabase instances, or external APIs. All validation happens within the local repository.
