# Backend Import Mapping

This document provides a high-level overview of the backend import mapping strategy for Caesar AI Scan.

For technical details, see the contract: [docs/backend/supabase/IMPORT_MAPPING_CONTRACT.md](docs/backend/supabase/IMPORT_MAPPING_CONTRACT.md).

The purpose of this mapping is to define how local scan findings and evidence packs are transformed for ingestion into the future backend storage system.
