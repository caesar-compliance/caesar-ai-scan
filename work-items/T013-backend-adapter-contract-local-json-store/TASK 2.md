# T013: Backend Adapter Contract and Local JSON Store Prototype

Implement a backend adapter contract and a local JSON store prototype to simulate future backend ingestion without adding real database/infrastructure.

## Deliverables
- `schemas/backend-adapter-store.schema.json`
- `src/backend-adapter/backend-adapter-contract.mjs`
- `src/backend-adapter/local-json-store-adapter.mjs`
- `src/backend-adapter/local-json-store-writer.mjs`
- `src/report/local-json-store-report.mjs`
- CLI support: `--local-store-dir`, `--local-store-report`, `--store-from-import`
